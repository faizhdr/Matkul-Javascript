Event onLoad

Event onload terjadi ketika sebuah objek telah dimuat. onload paling sering digunakan dalam elemen body untuk mengeksekusi skrip setelah halaman web memuat semua konten (termasuk gambar, file skrip, file CSS, dll.). Event onload dapat digunakan untuk memeriksa jenis browser pengunjung dan versi browser, dan memuat versi halaman web yang tepat berdasarkan informasi.
